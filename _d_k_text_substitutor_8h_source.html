78 /*
79 
80 This objects abstracts the text substitution task used by text adornments, et. al. It allows strings of the form:
81 
82  "This is fixed text %%sub1 more fixed text %%sub2 and so on..."
83 
84  Where %%sub1 and %%sub2 (where the word following %% represents a metadata key) are replaced by the metadata value keyed.
85 
86  A non-property key can also have further flags, called subKeys. These are . delimited single character attributes which invoke specific behaviours. By default these
87  are the digits 0-9 which extract the nth word from the original data, and the flags U, L and C which convert the data to upper, lower and capitalized strings respectively.
88 
89 */
