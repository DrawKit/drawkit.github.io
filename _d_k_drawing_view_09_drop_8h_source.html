25 Drag and Drop is extended down to the layer level by this category. When a layer is made active, the drawing view will register its
26 pasteboard types (because this registration must be performed by an NSView). Subsequently all drag/drop destination messages are

28 is interested in, just as if it were a view. The layer can use [self currentView] if it needs to access the real view object.

31 check itself.
32
