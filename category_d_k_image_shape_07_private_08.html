- (NSAffineTransform*) imageTransformWithoutLocation

return a transform that can be used to position, size and rotate the image to the shape

Note
a separate transform is necessary because trying to use the normal shape transform and rendering the image into a unit square results in some very visible rounding errors. Instead the image is transformed independently from its orginal size directly to the final size, so the errors are eliminated.
Returns
a transform
