75 /*
76 
77 This object implements an heuristic solution to the travelling salesman problem. The algorithm is based on simulated annealing
78 and is due to "Numerical Recipes in C", Chapter 10.
79 
80 To use, initialise with an array of NSValues containing NSPoints. Then request the shortestRoute. The order of points returned by -shortestRoute
81 will be the shortest route as determined by the algorithm. The first point object in both input and output arrays is the same - in other words
82 the zeroth element of the input array sets the starting point of the path.
83 
84 For uses with other object types, the -shortestRouteOrder might be more useful. This returns an array of integers which is the order of the
85 objects. This can then be used to reorder arbitrary objects.
86 
87 Most simply, the +sortedArrayOfObjects:byShortestRouteForKey: will deal with any objects as long as they have a KVC-compliant property that
88 resolves to an NSPoint return value, and is given by <key>. The result is a new array of the same objects sorted according to the TSP solution.
89 
90 */
