143 /*
144 
145 This category on NSBezierPath converts to and from the gpc_polygon data structure used by
146 the wonderful gpc (general polygon clipping) lib. This lib is used to perform boolean ops on paths.
147 
148 Note that at present paths are flattened into polygons and so curve control points, etc are not preserved.
149 
150 The curve-fitting is accomplished using 3rd party code from Lib2Geom, which is in turn a C++ implementation
151 of the classic Graphics Gems code. Curve-fitting is controlled by the "simplifying policy" that you set. By
152 default it's set to 'auto', meaning that if either of the original paths contains curves, curve fitting will
153 be done on the result, but if both source paths only have line segments, it won't be. This preserves sharp-cornered
154 shapes such as rects, etc.
155 
156 For simplifying a path at any other time, you must pass a flattened path. Simplifying really means "unflattening".
157
