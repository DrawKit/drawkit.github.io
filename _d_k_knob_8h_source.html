142 /*
143 
144 simple class used to provide the drawing of knobs for object selection. You can override this and replace it (attached to any layer)
145 to customise the appearance of the selection knobs for all drawn objects in that layer.
146 
147 The main method a drawable will call is drawKnobAtPoint:ofType:userInfo:
148 
149 The type (DKKnobType) is a functional description of the knob only - this class maps that functional description to a consistent appearance taking
150 into account the basic type and a couple of generic state flags. Clients should generally avoid trying to do drawing themselves of knobs, but if they do,
151 should use the lower level methods here to get consistent results.
152 
153 Subclasses may want to customise many aspects of a knob's appearance, and can override any suitable factored methods according to their needs. Customisations
154 might include the shape of a knob, its colours, whether stroked or filled or both, etc.
155 
156 */
