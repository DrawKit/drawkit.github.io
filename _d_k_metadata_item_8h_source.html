151 /*
152 
153  DKMetadataItems are used to store metadata (attribute) values in user info dictionaries attached to various objects such as layers and
154  drawables. Using a special wrapper preserves the type information under editing whereas using raw NSValue/NSNumber objects does not.
155 
156  Values passed to -setValue are always converted to the current type wherever possible. Conversely, using -setType converts the current value
157  to that type where possible. A conversion is always attempted, so in some cases a nonsensical conversion will result in data loss, e.g.
158  converting a URL to a colour. The -isLossyConversionToType: will return YES for lossy conversions, NO if the conversion will succeed.
159 
160  <type> and <value> properties are KVO-observable, any other methods call these.
161 
162  Values are stored in whatever class is appropriate to the type, viz:
163 
164  Type Class
165  ----------------------------
166  String NSString
167  Integer NSNumber (int)
168  Real NSNumber (double)
169  Boolean NSNumber (BOOL)
170  Unsigned NSNumber (int)
171  Attributed String NSAttributedString
172  Image NSImage
173  Image Data NSData
174  Data NSData
175  URL NSURL
176  Date NSDate
177  Size NSString
178  Point NSString
179  Rect NSString
180 
181  */
