79 /*
80 This is a very general-purpose view class that provides some handy high-level methods for doing zooming. Simply hook up
81 the action methods to suitable menu commands and away you go. The stuff you draw within drawRect: doesn't need to know or
82 care abut the zoom of the view - you can just draw as usual and it works.
83 
84 NOTE: this class doesn't bother to support NSCoding and thereby encoding the view zoom, because it usually isn't important for this
85 value to persist. However, if your subclass wants to support coding, your initWithCoder method should reset _scale to 1.0. Otherwise
86 it will get initialized to 0.0 and NOTHING WILL BE DRAWN.
