Detailed Description

The actual storage object. This inherits the linear array which actually stores the objects, but maintains a BSP tree in parallel, which stores indexes that refer to this array. Thus the objects' Z-order is strictly maintained by the array as for the linear case, but objects can be extracted very rapidly when performing a spatial query.
