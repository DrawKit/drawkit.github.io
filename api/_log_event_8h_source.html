33 // The LogEvent files contain functions (most notably LogEvent()) useful for
34 // *conditionally* logging various types of events or steps with in a process. The intent
35 // is for an application to allow an end user to enable (or "turn on") various types of
36 // logging (or permanent debugging). The logging of messages for any given type is
37 // prevented unless its type has been turned on (i.e., a user pref has been set).
38 // For you to use the LogEvent functionality, qUseLogEvent must be defined (typically
39 // in a precompiled header). If you want event logging to drop out of one of your
40 // configurations (or you do not agree to the licensing terms), don't define qUseLogEvent.
41 // (For more about dropping event logging out of one of your configurations, see the
42 // Macros section below.)
43 //
44 // The LoggingController class encapsulates an NSWindowController for modifying the
45 // user selectable logging options. It follows the singleton design pattern and therefore
46 // follows Apple's guidelines for a singleton (cf. "Creating a Singleton Instance" chapter
47 // in the Cocoa Fundamentals Guide).
48 //
49 // Although frameworks may use LogEvent() and the various event types defined below,
50 // they generally do not actually make use of the LoggingController class (and its nib
51 // file). The LoggingController class is a UI level object that simplifies turning logging
52 // on or off. Adding it to your application is easy.
53 // In our projects, we typically have access to the Logging dialog "hidden" in the
54 // -showAboutBox: IBAction. In other words, to enable or disable logging, a user simply
55 // holds down a modifier key while choosing About Application. It looks something like
56 // this:
57 
58 /*- (IBAction)showAboutBox:(id)sender
59 {
60  BOOL isOptionKeyDown = (([[NSApp currentEvent] modifierFlags] & NSAlternateKeyMask) != 0);
61  if (isOptionKeyDown)
62  {
63  [[LoggingController sharedLoggingController] showLoggingWindow];
64  }else
65  {
66  [[AboutBox sharedAboutBox] showAboutBox:sender];
67  }
68 }*/
