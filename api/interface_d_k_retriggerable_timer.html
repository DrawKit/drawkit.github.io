This software is released subject to licensing conditions as detailed in DRAWKIT-LICENSING.TXT, which must accompany this source file.Implements a one-shot timer that can be repeatedly extended (retriggered) preventing it timing out. When it does time out, it calls the target/action. It can be retriggered to start a new cycle after timing out.

This is analogous to a retriggerable monostable in electronics - useful for detecting when a series of rapid events ceases if there is no other way to detect them. Each event calls -retrigger, extending the timeout until no more retriggers + the period elapses.
