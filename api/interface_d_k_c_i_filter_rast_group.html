Detailed Description

Captures the output of its contained renderers in an image.

Author
Contributions from the community; see CONTRIBUTORS.md
Date
2005-2015
Copyright
GNU LGPL3; see LICENSEThis class implements a special rendergroup that captures the output of its contained renderers in an image, then allows that image to be manipulated or processed (e.g. by core image) before rendering it back to the drawing. This allows us to leverage all sorts of imaging code to extend the range of available styles and effects.
