140 This class provides a simple hatching fill for a path. It draws equally-spaced solid lines of a given thickness at a
141 particular angle. Subclass for more sophisticated hatches.

144

146 than the cached size, it is not rebuilt. It is rebuilt if the angle or spacing changes or a bigger path is hatched. Linewidth also
147 doesn't change the cache.
