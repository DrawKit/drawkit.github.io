713 /*
714 
715 drawing layers are lightweight objects which represent a layer. They are owned by a DKDrawing which manages the
716 stacking order and invokes the drawRect: method as needed. The other state variables control whether the layer is
717 visible, locked, etc.
718 
719 DKDrawing will not ever call a drawRect: on a layer that returns NO for visible.
720 
721 if isOpaque returns YES, layers that are stacked below this one will not be drawn, even if they are visible. isOpaque
722 returns NO by default.
723 
724 locked layers should not be editable, but this must be enforced by subclasses, as this class contains no editing
725 features. However, locked layers will never receive mouse event calls so generally this will be enough.
726 
727 As layers are retained by the drawing, this does not retain the drawing.
728 
729 By definition the bounds of the layer is the same as the bounds of the drawing.
730 
731 */
