198 /*
199 
200 drawing layers are lightweight objects which represent a layer. They are owned by a DKDrawing which manages the
201 stacking order and invokes the drawRect: method as needed. The other state variables control whether the layer is
202 visible, locked, etc.
203 
204 DKDrawing will not ever call a drawRect: on a layer that returns NO for visible.
205 
206 if isOpaque returns YES, layers that are stacked below this one will not be drawn, even if they are visible. isOpaque
207 returns NO by default.
208 
209 locked layers should not be editable, but this must be enforced by subclasses, as this class contains no editing
210 features. However, locked layers will never receive mouse event calls so generally this will be enough.
211 
212 As layers are retained by the drawing, this does not retain the drawing.
213 
214 By definition the bounds of the layer is the same as the bounds of the drawing.
215 
216 
217 */
