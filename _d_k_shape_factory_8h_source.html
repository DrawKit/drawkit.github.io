84 /*
85 
86 This class provides a number of standard shareable paths that can be utilsed by DKDrawableShape. These are all
87 bounded by the standard unit square 1.0 on each side and centered at the origin. The DKDrawableShape class
88 provides rotation, scaling and offset for each shape that it draws.
89 
90 Note that for efficiency many of the path objects returned here are shared. That means that if you change a shape
91 with the path editor you MUST copy it first.
92 
93 The other job of this class is to provide shapes for reshapable shapes on demand. In that case, an instance of
94 the shape factory is used (usually sharedShapeFactory) and the instance methods which conform to the reshapable informal
95 protocol are used as shape providers. See DKReshapableShape for more details.
