115 /*
116 
117 This category provides methods for exporting drawings in a variety of formats, such as TIFF, JPEG and PNG. As these are all bitmap formats,
118 a way to specify the resolution of the exported image is also provided. All methods return NSData that is the formatted image data - this can be
119 written directly as a file of the designated kind.
120 
121 All image export starts with the pdf representation of the drawing as exported directly by DKDrawing. This is then imaged into a new bitmap image
122 rep before conversion to the final format. The use of the pdf data ensures that results are consistent and require no major knowledge of the
123 drawing's internals.
124 
125 All images are exported in 24/32 bit full colour.
126 
127 dpi is specified directly, e.g. 72 for 72 dpi, 150 for 150 dpi, etc. The image size will be the drawing size scaled by the dpi, so a 144dpi image
128 will be twice as wide and twice as high as the drawing. If the dpi passed does not result in a whole multiple of the drawing size, it is rounded up
129 to the nearest whole value that is.
130 
131 This uses Image I/O to perform the data encoding.
132 
133 */
