37 /*
38 
39 Most drawables in DK have contextual menus associated with them. This objects allows those menus to be simply defined in a nib
40  file in the framework, and overridden by a similar file in the host app. This object supplies one menu per object class.
41 
42  The default nib is set up so that the menus target first responder, such that DK's message forwarding handles menu validation as
