209 /*
210 
211 DKDrawableShape is a semi-abstract base class for a drawable object consisting of any path-based shape that can be drawn to fit a
212 rectangle. DKShapeFactory can be used to supply lots of different path shapes usable with this class.
213 
214 This implements rotation of the shape about a specified point (defaulting to the centre), and also standard selection handles.
215 
216 Resizing and moving of the shape is handled by its implementations of the mouseDown/dragged/up event handlers.
217 
218 This class uses the handle drawing supplied by DKKnob.
219 
220 The path is stored internally in its untransformed form. This means that its datum is at the origin and it is unrotated. When rendered, the
221 object's location and rotation angle are applied so what you see is what you expect. The bounds naturally refers to the transformed
222 bounds. The selection handles themselves are also transformed by the same transform, since the shape remains editable at any orientation.
223 
224 The canonical path is set to have a bounding rect 1.0 on each side. The actual size of the object is factored into the transform to
225 render the object in the drawing. Thus the original path is NEVER changed once established. This allows us to share basic shapes which
226 can be generated by a factory class.
227 
228 
229 */
