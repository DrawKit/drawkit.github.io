90 /*
91  DKRasterizer is an abstract base class that implements the DKRasterizer protocol. Concrete subclasses
92  include DKStroke, DKFill, DKHatching, DKFillPattern, DKGradient, etc.
93 
94  A renderer is given an object and renders it according to its behaviour to the current context. It can
95  do whatever it wants. Normally it will act upon the object's path so as a convenience the renderPath method
96  is called by default. Subclasses can override at the object or the path level, as they wish.
97 
98  Renderers are obliged to accurately return the extra space they need to perform their rendering, over and
99  above the bounds of the path. For example a standard stroke is aligned on the path, so the extra space should
100  be half of the stroke width in both width and height. This additional space is used to compute the correct bounds
101  of a shape when a set of rendering operations is applied to it.
102 
103 */
